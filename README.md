# Cricbuzz LiveStats

## Project Overview
A real-time cricket analytics dashboard using API, SQL, and Streamlit.

## Features
- Live match updates (API integration)
- Player statistics dashboard
- SQL analytics (25 queries)
- CRUD operations for database

## Tech Stack
- Python
- Streamlit
- SQLite
- REST API
- Pandas

## How to Run
```bash
pip install -r requirements.txt
python create_tables.py
python insert_data.py
streamlit run app.py
